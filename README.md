# Infinite-Canvas

一个基于 Web 的无限画布 AI 创作工具，支持文生图、图生图、视频生成、LLM 对话、ComfyUI 工作流等多种 AI 能力，所有功能以节点方式可视化编排。

## 快速开始

1. 安装依赖：双击运行 `安装依赖.bat`（Windows）；换电脑时会自动下载便携版 Python，**无需手动安装 Python**
2. 启动：双击运行 `run.bat` 或执行 `python main.py`
3. 打开浏览器访问 `http://localhost:3000`

## API 配置

在网页内直接进入设置页面配置 API Key，支持：
- OpenAI 协议（GPT、Claude、Gemini 等）
- ModelScope（通义万象等）
- 自定义 API 地址
